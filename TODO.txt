[] сделать страницу Messages

[] Основной функционал:
	[] Отображение списка сообщений:
		[]Сообщения в виде карточек или строк.
		[]Отображение отправителя, текста сообщения, времени отправки.
	
	[] Загрузка сообщений:
		[] Пагинация (постраничная загрузка).
		[] Ленивая загрузка (infinite scroll).
  
	[] Отправка сообщений:
		[] Поле ввода сообщения с кнопкой "Отправить".
		[] Отправка сообщений через API.
		[] Валидация сообщения (например, ограничение по длине).
	
	[] Реалтайм обновления:
		[] Подключение WebSocket или SSE для получения новых сообщений.
		[] Автоматическое обновление списка.

  [] Поиск и фильтрация:
		[] Поиск сообщений по тексту.
		[] Фильтрация по пользователю, дате, статусу (прочитанные/непрочитанные).
	
	[] Сортировка сообщений:
		[] По дате (сначала новые/старые).
		[] По имени отправителя (если актуально).

	[] Статусы сообщений:
		[] Прочитанные и непрочитанные сообщения.
		[] Отображение индикатора новых сообщений.
	
	[] Действия с сообщениями:
		[] Удаление сообщений.
		[] Архивация сообщений.
		[] Возможность отметить сообщение как прочитанное/непрочитанное.
	
	[] Интеграция с аватарами пользователей:
		[] Отображение аватара отправителя рядом с сообщением.
		[] Возможность перейти на профиль отправителя.

	[] Обработка ошибок:
		[] Показ уведомлений при ошибке загрузки или отправки.
		[] Повторная попытка отправки сообщения.
	
[] Дополнительный функционал:
	[] Push-уведомления:
		[] Уведомления о новых сообщениях даже при неактивной вкладке.
	
	[] Адаптация под мобильные устройства:
		[] Респонсивный дизайн.
		[] Удобный интерфейс для мобильных экранов.
	
	[] Групповые чаты:
		[] Поддержка отображения чатов с несколькими участниками.
		[] Вывод информации о количестве участников.
	
	[] Вложения:
		[] Возможность отправки изображений, документов или других файлов.
		[] Превью вложений в списке сообщений.

	[] Темизация:
		[] Поддержка светлой и тёмной темы.
	
	[] Интернационализация:
		[] Локализация сообщений, кнопок и других элементов на разные языки.
